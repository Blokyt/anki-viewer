# ANKI Viewer

Application web pour visualiser et gérer les cartes Anki.

## Fichiers du projet

- `index.html` - Page principale de l'application
- `styles.css` - Styles de l'interface
- `app.js` - Logique JavaScript de l'application
- `anki_to_json.py` - Script Python pour convertir les collections Anki en JSON

## Utilisation

1. Ouvrez `index.html` dans votre navigateur
2. Utilisez le script Python pour convertir vos collections Anki si nécessaire

## Notes

Les fichiers `.colpkg` (collections Anki) sont exclus du dépôt Git en raison de leur taille importante.
